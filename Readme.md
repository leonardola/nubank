## Projeto Nubank

```pip install pymysql
pip install peewee
pip install pyyaml
pip install pandas
```
Copie o arquivo config.yml.dist para config.yml e troque as credenciais 
Para executar a importação:

```python3.5 Importer.py```

Para executar o servidor

```python3.5 Index.py```

Agora acesse 

```localhost:5000```
